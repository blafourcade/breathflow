# BreathFlow — product overview

BreathFlow is a guided breathwork mobile app (iOS + Android) with:
- A breath timer for inhale / hold-full / exhale / hold-empty.
- 10+ curated presets (Box, 4-7-8, Wim Hof, Coherent, Bhastrika, Nadi Shodhana, Triangle, Buteyko, Physiological Sigh, Tactical).
- Custom pattern builder with share codes.
- Progression dashboard (streaks, total minutes, BOLT trend, longest hold).
- Friends, clans, leaderboards.

**Vision**: make intentional breathing as habit-forming as fitness tracking, as personal as a coach, as social as a fitness app.

**Target users**: optimisers (Huberman crowd), stressed professionals, athletes, yogis.

**Non-goals (v1)**: paid teacher courses, BLE HRV streaming, watch companion, AI coach.
