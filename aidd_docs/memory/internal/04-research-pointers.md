# Research pointers

- **Breathwork knowledge base**: `docs/breathwork_research.md` (16 protocols, physiology, performance tests, safety contraindications, audio cue patterns, progression metrics, citations).
- **BOLT scale**: <10 very low, 10-20 low, 20-30 moderate, 30-40 good, 40+ excellent — implemented in `packages/breath-engine/src/bolt.ts`.
- **Resonance breathing**: 5.5 bpm (5.5s in / 5.5s out) maximises HRV — `coherent-5.5` preset.
- **Hard contraindications**:
  - Pregnancy, epilepsy, cardiovascular disease, glaucoma, severe psychiatric: avoid retention-heavy protocols.
  - Wim Hof: NEVER in water, NEVER while driving.
- **App-survey notes**: Breathwrk / Othership / Calm cue styles documented in `docs/breathwork_research.md` §5.
